## Prática de Python

Esse repositório é dedicado a pratica da Linguagem de programação Python, sendo ela realizada por uma bateria de exercícios.
Aqui foi trabalhado coisas como:
- Lógica de programação
- Variáveis
- Condições
- Listas, tuplas, dicionários
- Estruturas de Repetição
- Loop Infinito
- Funções
- POO
A pratica realizada nesse reposítorio irá se manter ad aeternum.
